# Lista de Chequeo GLOBALG.A.P.

App en Streamlit para completar una lista de chequeo basada en un Excel fijo.

## Archivos

- `app.py`: aplicación principal.
- `norma_globalgap.xlsx`: Excel base con la norma.
- `requirements.txt`: dependencias para Streamlit Cloud.

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Despliegue en Streamlit Cloud

1. Crear repositorio en GitHub.
2. Subir estos tres archivos.
3. Entrar a Streamlit Cloud.
4. Crear nueva app desde el repositorio.
5. Seleccionar `app.py` como archivo principal.
6. Deploy.
